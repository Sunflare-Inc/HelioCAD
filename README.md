# HelioCAD

HelioCAD is a lightweight 2D CAD tool made in Python (snek 🐍) using PyQt6.

### Features:
- Draw lines, rectangles, circles
- Snap to grid
- Save/load `.dcad` files
- Export SVG vector graphics

More coming soon... 👀

### Run it:
```bash
pip install PyQt6 svgwrite
python HelioCAD.py
